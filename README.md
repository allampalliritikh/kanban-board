# 🗂️ MEVN Kanban Board

A full-stack Kanban Board application built using the **MEVN Stack (MongoDB, Express.js, Vue.js, Node.js)** that allows users to create, organize, and manage tasks visually using draggable boards.

This project demonstrates modern full-stack development practices including REST APIs, component-based frontend architecture, and database integration.

---

# 🚀 Features

* Create and manage tasks
* Drag and drop tasks across columns
* Organize tasks using Kanban workflow
* Real-time UI updates
* RESTful API integration
* Responsive frontend interface
* Modular and scalable project structure

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/allampalliritikh/kanban-board.git
cd kanban-board
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

Backend will run on:

```
http://localhost:5000
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

---

# 📸 Application Workflow

1. User creates tasks
2. Tasks appear in Kanban columns
3. Tasks can be moved between columns
4. Backend API stores and retrieves task data from MongoDB
